This is a battleships game between user and computer.
The board size is 10x10
Gameplay:
1.user deployes 5 ships giving appropriate coordinates.
2.computer deployes 5 ships.
3.user move - try to guess the location of one of computer ships. Possible outcomes: miss, sunk computer ship, sunk your own ship.
4.computer move - computer randomly picks a location. Possible outcomes: miss, sunk its own ship, sunk user ship.
5.game ends when computer ships are all sunk (user wins) or user ships are all sunk (computer wins).
